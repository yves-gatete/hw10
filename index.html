Information about this page

Description: This API allows its users access data on COVID-19. The data displayed is from the repository for the 2020 Novel Coronavirus Visual Dashboard operated by the Johns Hopkins University Center for Systems Science and Engineering (JHU CSSE). Also, Supported by ESRI Living Atlas Team and the Johns Hopkins University Applied Physics Lab (JHU APL).It gets its data from Johns Hopkins CSSE

API site: A free API for data on the Coronavirus

Options used: I used the "GET" option to retreive data from the server at the specified resource. Data can be retrieved from the API using a lot of options including id, country, country code, new confirmed etc. For my GET request, I used the filter to a few specific fields mainly: New Cases, Total Cases, New Deaths, Total Deaths, New Recoveries, and Total Recoveries.

Useful applications: For health-related organizations, pharmaceutical companies, businesses, and even personal use.
